Connexion Instagram
Connecte ton compte pour publier automatiquement
Pour publier automatiquement sur Instagram, tu as besoin d'un compte Business ou Creator et d'un token Meta API. Suis les étapes ci-dessous.
Étape 1 — Compte Instagram

Ton compte doit être en mode Business ou Creator. Si c'est un compte personnel, voici comment changer :

  1. Ouvre Instagram → ton profil
  2. Paramètres → Compte
  3. "Passer en compte professionnel"
  4. Choisir Business ou Creator
Mon compte est actuellement :
Étape 2 — Page Facebook

L'API Meta exige une Page Facebook liée à ton compte Instagram. C'est obligatoire, même si tu ne l'utilises pas.

  1. Va sur facebook.com/pages/create
  2. Crée une page "L'information IA"
  3. Dans Instagram → Paramètres → Compte lié → Page Facebook
Créer une Page Facebook →
Étape 3 — Obtenir ton Token Meta API
1
Va sur developers.facebook.com → "Mes apps" → Créer une app
2
Choisis le type "Business" → donne un nom (ex: Agent IA Instagram) → Créer
3
Dans ton app → Produits → ajoute "Instagram Graph API"
4
Va dans Outils → Explorateur d'API Graph → génère un token avec les permissions : instagram_basic, instagram_content_publish, pages_read_engagement
5
Copie le token et colle-le ci-dessous
Trouve-le dans : Instagram → Paramètres → À propos → ID du compte
Instagram connecté avec succès ! Ton agent peut maintenant publier automatiquement sur ton compte.
